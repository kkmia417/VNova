# Open Decision Register

Status: Requires human decisions

OPEN decisions are not production defaults. Each entry names an accountable human role and the implementation scope it blocks so unrelated work can continue safely. Named individuals are assigned during protected human review.

An item remains OPEN unless it has a valid entry in the
[Open Decision Disposition Register](../governance/open-decision-dispositions.md). The only
exception is OD-040's one-time genesis ratification, whose pre-existing human authority and
evidence are defined in that register. An ADR status change, a blank review row, meeting notes, or
an agent-authored recommendation is not a disposition.
